# shapez 2 api

[api docs](https://shapez2api.raphdf201.net/v1/openapi)

server config format : put a file named config in the working dir of the server

```
apikey          // 0
dburl           // 1
dbuser          // 2
dbpw            // 3
workshopdlpath  // 4
steamuname      // 5
steamcmdpath    // 6
```
